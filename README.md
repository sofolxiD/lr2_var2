 2) Класс Гараж хранит данные об автомобилях, мотоциклах и автобусах. Для авто. опред.: марка, модель, объем двиг., цвет, тип КПП. Для мото. опред: марка, модель, объем двигателя, мощность двиг., для какой местности предназначен. Для автоб. опред.: марка, модель, колич. сид. пассажирских мест, общее количество пассажирских мест, конечный пункт.
